| **APA102** (DotStar) | SPI | 500+ | Höhere Refresh-Rate, Global Brightness |

**Raspberry Pi Features:**
- Alle Pi-Modelle (Pi 5, Pi 4, Pi 3, Pi Zero 2W, ...)
- GPIO, SPI, I2C, PWM Unterstützung
- LED Matrix Support (HUB75)
- systemd-Autostart
- Remote-Steuerung über SSH/API/MQTT

## Protokoll-Übersicht

| Protokoll | Beschreibung | Vorteile |
|-----------|--------------|----------|
| **HID** | USB Human Interface Device | Universell, treiberlos, Cross-Platform |
| **SMBus** | System Management Bus | Direkter RAM-Zugriff, Low-Latency, DDR5 |
| **SPI** | Serial Peripheral Interface | Schnell, für lange LED-Strips |
| **GPIO** | General Purpose I/O | Raspberry Pi, Maker-Projekte, DIY |
